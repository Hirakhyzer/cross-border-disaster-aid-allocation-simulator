# Synthetic Aid Allocation Lab

Run:

```bash
python scripts/run_synthetic_aid_lab.py
```

Optional:

```bash
python scripts/run_synthetic_aid_lab.py --regions 28 --warehouses 5 --seed 42 --policy fairness_aware
```

The lab creates fictional regions, warehouses, inventory, routes, needs, allocation plans, unmet-need tables, fairness audits, logistics summaries, policy comparisons, reports, and figures.

## Interpretation rules

- Results are synthetic and local.
- The allocation plan is not an official humanitarian plan.
- Fairness metrics are audit signals for research comparison.
- Route-delay estimates are simplified simulations, not navigation or customs predictions.
- Real use requires humanitarian partner validation and governance review.
